# CloudAssignment

This Web Application is to find whether the entered number is prime or not. In the Result window you will be able to check whether the number is prime or not.

<H> Rest API </h>
The Application has the REST API endpoint where it can be accessible through REST Client Like the Postman.
The Application is Developed Using the Jesey Framework in java .

Technologies Used: Servlets,Java,HTML,Java Script and BootStrap

Server used : Apache tomcat 8

Public URL : ec2-54-244-144-50.us-west-2.compute.amazonaws.com/CloudProject/
Rest API URL : ec2-54-244-144-50.us-west-2.compute.amazonaws.com/CloudProject/api/prime?number=11&name=sam
